Land packagers can create enormous property value increase by annexation agreements, zoning changes, and conceptual site plan approvals.

  • Each zoning designation has permitted uses.
  • Easier to change the permitted use language than to change the zoning.
  • Land use zoning is traditional.
  • Urban envelope zoning is becoming popular.

These approvals set the stage for future development, so they must be flexible yet clear.

  • Often design guidelines are part of the entitlement documents.
  • Developer needs the flexibility to respond to changing market conditions.

Every approval that the land packager gets increases the value of the property to the land developer.

  • Unless the wrong land uses are approved.
  • Unless there is no flexibility in the approvals.
